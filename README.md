Power BI Ecosystem: Business Intelligence Portfolio
Este repositorio contiene una colección de proyectos avanzados de Power BI (.pbix). Cada dashboard está diseñado para resolver retos específicos de negocio, transformando datos complejos en visualizaciones interactivas y accionables.

Proyectos Incluidos
1. Análisis de Ventas de Electrodomésticos
Dashboard enfocado en el sector Retail, optimizado para el seguimiento de rendimiento de productos y márgenes.

KPIs Clave: Margen de contribución, cumplimiento de objetivos de ventas y rotación de stock.

Habilidades: Modelado de datos relacionales, análisis de categorías y segmentación de productos.

2. Análisis Financiero Multi-Banco
Visualización estratégica para el sector bancario que permite comparar métricas entre diferentes instituciones.

KPIs Clave: Volumen de transacciones, análisis de depósitos vs. créditos y perfiles de riesgo.

Habilidades: DAX avanzado para comparativas entre bancos y seguridad a nivel de fila (RLS) potencial.

3. Monitor de Productividad de Empleados
Un enfoque de HR Analytics para entender el rendimiento del capital humano a través de diferentes dimensiones.

KPIs Clave: Eficiencia por región, cumplimiento de metas por mes y rankings de empleados.

Habilidades: Inteligencia de tiempo (Time Intelligence) para ver evolución mensual y filtros cruzados por jerarquías geográficas.

4. Modelo de Datos de Ventas (Integral)
Análisis técnico profundo de un ciclo de ventas completo, ideal para detectar cuellos de botella en el embudo comercial.

KPIs Clave: Ticket promedio, tasa de retención de clientes y análisis de cohortes.

Habilidades: Esquema en estrella (Star Schema) robusto y optimización del motor VertiPaq.
